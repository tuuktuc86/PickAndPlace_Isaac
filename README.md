# Efficient DAgger for Failure Driven Robot Learning

This repository contains information related to the (paper): **Balanced DAgger: Safe Online Imitation Learning for Robot Manipulation with Direction Balanced RND and Conformal Prediction** and (project)**Efficient DAgger for Failure Driven Robot Learning**.


## Notice on Code Availability

Please note that the core source code is not publicly available. As this project is part of an ETRI-sponsored research project, we do not disclose the core algorithms and implementation details to comply with the institution's confidentiality guidelines.
